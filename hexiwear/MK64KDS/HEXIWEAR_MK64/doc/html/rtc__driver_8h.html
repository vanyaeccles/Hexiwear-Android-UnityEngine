void RTC_AlarmCallback ( )

interrupt callback

RTC interrupt handler callback

RTC also puts MCU to sleep

if we're in normal mode, proceed to go to sleep

in case of the shallow and total sleep mode, trigger the sleep task to finish the procedure

for the sensor tag sleep mode, announce that we're ready for sensor tag sleep mode

this shouldn't happen, if RTC is enabled, we shouldn't be sleeping
